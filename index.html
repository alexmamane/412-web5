Just Do It App

Project Goal

This project was meant to create a simple to do list app for teachers. This app will allow the users to...

  • - Create an account using a username
  • - Log in or identify ourselves in the system with our username
  • - Display the list of our individual “to do” tasks
  • - Display a list of our 10 last completed tasks
  • - Provide a simple text field to add a new task to the list
  • - Provide a checkbox or a button to be able to mark a pending task as complete

Challenges

  • - Intergeating the database
  • - Using the exacte words

Sometimes we spelled words differently like words with an (s) and sometimes w without so it took us forever to finally realize and then it finally connected to the db and would show data
